1. 两数之和

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素在答案里不能重复出现。

你可以按任意顺序返回答案。

提示：

  • 2 <= nums.length <= 104
  • -109 <= nums[i] <= 109

进阶：你可以想出一个时间复杂度小于 O(n2) 的算法吗？

因为题目假设每种输入只会对应一个答案，且数组中同一元素在答案中不能重复出现，故考虑使用哈希表来进行存储

整体思路：

给出目标 target 求是否在数组中存在两数，使其和等于 target。本质上可以转化为，在遍历数组的过程中，每遍历一个数组元素，判断是否存在 target - nums[i] 于数组中，若存在则返回当前遍历数组元素的位置 i 与其对应的余数 target - nums[i] 的位置。已知数组元素不重复，且答案只存在一个，所以可以使用 哈希表 进行存储。将数组元素作为 key，数组元素对应下标作为 value，在便于判断对应元素是否存在的同时，可以使用 get() 函数由 key 很轻易的得到对应的 value。
