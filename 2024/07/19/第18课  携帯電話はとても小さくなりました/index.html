第18课 携帯電話はとても小さくなりました

第18课 携帯電話はとても小さくなりました

基本课文

風邪はどうですか。

昨夜薬を飲みましたが，までよくなりません。

旅行の荷物は軽いほうがいいですよ。

补充说明：关于 ほうが 的具体用法，本身是“更加的意思”，这里组合 ほうがいい 表示“更好”，在本句的意思即为“行李越轻越好。”

在第12課中，我们学习了：

  1. 名1 + より + 名2 + のほうが + 一类形/二类形 です

    和 名1 相比，名2 更 xx。

  2. 名1 + は + 名2 + より + 一类形/二类形 です

    名1 比 名2 更xx。

  3. 名1 + は + 名2 + ほど + 一类形 くないです / 二类形 ではありません

    名1 不及 名2 xx。

    注意，此处涉及形容词的否定形式，之前为提到过，在此补充：一类形容词的否定形式的构成方法如下：

    一类形容词（形容词本身结尾带有“い”）

    1. 去掉形容词的“い”。
    2. 加上“くない”。

    例如：

    • 寒い (さむい) → 寒くない (さむくない) - “不冷”
    • 高い (たかい) → 高くない (たかくない) - “不高”
    • 新しい (あたらしい) → 新しくない (あたらしくない) - “不新”

    二类形容词（形容动词，通常用“な”连接）

    二类形容词的否定形式的构成方法如下：

    1. 使用“じゃない”或“ではない”来表示否定。

    例如：

    • 静か (しずか) → 静かじゃない (しずかじゃない) - “不安静”
    • 便利 (べんり) → 便利じゃない (べんりじゃない) - “不方便”
    • きれい → きれいじゃない - “不漂亮”

はい。できるだけ軽くします。

补充说明：为什么这个句子中用 軽く（连用形，可以理解为去ます形） 而不是 軽い？

在日语中，形容词的“く”形式通常用于连接动词或其他形容词，表示某种状态或方式。在“できるだけ軽くします”这句话中，形容词“軽い”变成了“軽く”，这是为了连接动词“します”。

详细解释

  1. できるだけ：
    • 副词短语，意思是“尽量、尽可能”。
  2. 軽く：
    • 形容词“軽い” (かるい) 的连用形。
    • 形容词的连用形是通过将“い”变成“く”来构成的，这样可以连接动词，表示某种动作是在这种状态或方式下进行的。
    • 在这里，“軽く”表示“轻地”。
  3. します：
    • 动词“する”的敬体形式，意思是“做”。

句子结构分析

  • できるだけ (尽可能地)【副词短语】
  • 軽く (轻地)【形容词的连用形】
  • します (做)【动词】

这句话的意思是“尽量把它弄轻”，其中“軽く”表示动作“します”的方式是“轻地”。

形容词连用形的其他用法

形容词连用形（“く”形）有几种常见用法：

  1. 连接动词：
    • 表示某个动作是在某种状态或方式下进行的。
    • 例如：“速く走る” (はやくはしる) - 快速地跑。
  2. 表示原因（通常在文言文或书面语中使用）：
    • 例如：“忙しくて会えない” (いそがしくてあえない) - 因为忙而不能见面。
  3. 副词化：
    • 形容词的“く”形可以直接变成副词，表示一种方式或程度。
    • 例如：“静かに” (しずかに) - 安静地。

お嬢さんはおいくつですが。

今年七歳になりました。

あのう，ちょうとうるさいですよ。

あっ，すみません。静かにします。

语法解释

  1. 一类形 + なります　（自动词代表）

​ 表示性质或者状态的变化。构成方法为将一类形的い变为く加なります。

​ だんだん暖かくなります。

​ 天气渐渐变暖。

  1. 一类形 + します　（他动词代表）

​ 用于主语的意志性动作、作用而引起事物发生变化的场合。构成方法是把一类形容词尾的 い 变成 く然后加します。

​ テレビの音を大きくします。

​ 把电视机的声音开大一点。

  1. 二类形 / 名词 になります

​ 同1，都是表示将某个东西变成 形容词或者名词 所表示的那个状态。只不过，一类形变连用形是将い变く加なります，而二类形或名词变连用形的时候，是直接在其后面接に（其实名词没用连用形，动词加に也是类似于形容词的连用形表示变成某种状态或结果，所以这里并列陈述）。

​ もう元気になりました。

​ 已经恢复健康了。

​ 息子は医者になりました。

​ 儿子要当医生了。

  1. 二类形 / 名词 + にします

​ 同2，表示因主语的意志性动作、作用等引起的变化。

​ 部屋を綺麗にしてください。

​ 请把房间打扫干净。

​ 会議室を禁煙にします。

  1. 一类型 / 二类形 + ほうがいいです

​ 在比较两个事物的性质，说明其中一个更好时，可用：

​ 一类形容词 + ほうがいいです

​ 二类形容词 + な + ほうがいいです

​ 旅行の荷物は軽いほうがいいです。

​ 旅行行李还是轻一点好。

​ 子供は元気なほうがいいです。

​ 孩子还是精神一点好。

  1. 自动词和他动词

​ 他动词具有主语作用于宾语而使其发生变化的含义，其宾语用 を 表示。自动词是不带宾语的动词，表示不考虑外力影响的情况下，主语自主进行动作或自然发生变化，其主语用が表示。

​ 窓が開きました。

​ 窗户开了。

​ （李さんが）窓を開けました。

​ 小李打开窗户了。

​ 注：带を即有作用于宾语的一定是他动词。
